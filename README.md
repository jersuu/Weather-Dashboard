# Weather Dashboard

Веб-приложение для получения актуальной погоды по названию города с использованием API сервиса OpenWeather.

<img width="850" height="600" alt="image" src="https://github.com/user-attachments/assets/3d9e871a-8e12-4b75-87e3-acb13ea43f7a" />
<br><br>
Возможности:
<br><br>
<ul>
  <li>Поиск по названию города</li>
  <li>Отображение температуры и погодных условий</li>
  <li>Прогноз погоды на 5 дней вперед</li>
  <li>Сохранение городов для пользователя с возможностью очистки</li>
</ul>
